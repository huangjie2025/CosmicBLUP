# CosmicBLUP 1.0.0 Release Notes

CosmicBLUP 1.0.0 establishes a dedicated BLUP codebase derived from the mature BLUP pipeline in CosmicSolver. Bayesian and variational Bayesian models are intentionally absent.

Highlights include PBLUP, GBLUP, ssGBLUP, repeatability, random regression, multi-trait GBLUP, multiple VCE strategies, relationship matrix workflows, QC, SBLUP, and prediction. The project introduces a responsibility-based source layout, vendored numerical core, portable tests, consistent product naming, and reproducible packaging definitions.

Known boundaries are documented in `MODEL_SUPPORT.md`.
